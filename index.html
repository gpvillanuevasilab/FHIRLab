Services

You can use FHIR Lab's services to experience and experiment with FHIR technology. Click to play! We offer

  1. Clinical Data Repository: FHIR Lab offers an instance of HAPI, an Open-Source FHIR server implementation. This server allows to experience basic features of FHIR and learn by practice.
  2. Terminology Server: FHIR Lab has a licence for Ontoserver, a high performance FHIR terminology server implementation. This server allows the reliable use and mapping of medical terminology and comes with excellent authoring tools.
