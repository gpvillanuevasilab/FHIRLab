FHIR Lab is an interoperability reference infrastructure (sandbox), co-developed by UPM SILab and CSIRO, Australia as part of the Strengthening Standards Capability Project (SSCP) for improved adoption of data standards and interoperability in the Southeast Asia and Pacific region​.

Services

You can use FHIR Lab's services to experience and experiment with FHIR technology. Click to play! We offer

  1. FHIR Server: Complete Reference Implementation of HL7 FHIR standards using the localised FHIR resources for testing and validation as part of the healthcare interoperability training and development activities.
  2. Terminology Server: Reference implementation of terminology using Ontoserver, a Next-gen FHIR terminology server developed by the Australian e-Health Research Centre, CSIRO”.
  3. FHIR Validator: an instance of the FHIR Validator App which allows the use of the FHIR Validator from the internet.
