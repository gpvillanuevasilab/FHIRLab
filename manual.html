FHIR Lab is an interoperability reference infrastructure deployed for the training, testing and validation of FHIR resources, terminology and Implementation Guides in a non-prod, secure, proof-of-concept (POC) environment. FHIR Lab is co-developed by UPM SILab and CSIRO - Australia's National Science Agency, as part of the CSIRO Strengthening Standards Capability Project (SSCP).

Terminology Server

Reference implementation of terminology using Ontoserver, a Next-gen FHIR terminology server developed by the Australian e-Health Research Centre, CSIRO

FHIR Server

Complete Reference Implementation of HL7 FHIR standards using the localised FHIR resources for testing and validation as part of the healthcare interoperability training and development activities

FHIR Validator

The validator will assist developers in checking the correctness of their FHIR implementations and validating that resources are conformant to the FHIR specifications.
